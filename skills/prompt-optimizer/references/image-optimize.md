# 图像优化

文生图和图生图提示词优化。

## 文生图优化 (text2imageOptimize)

### 1. 通用自然语言优化 (通用)

**适用场景**: 大多数图像生成需求 (Midjourney, Stable Diffusion, DALL-E 等)

**触发方式**:
```
/image 一只猫
图像：未来城市
```

**优化方向**:
- 主体明确（主体是什么）
- 动作/状态（做什么）
- 环境锚点（在哪里）
- 光线（光照条件）
- 配色（色彩方案）
- 材质（表面质感）
- 氛围（情绪风格）
- 构图视角（镜头角度）

---

### 2. 中文工具优化 (即梦 / 讯飞等)

**适用场景**: 中文图像生成工具（如即梦、讯飞星火等）

**触发方式**:
```
/image 一只猫
图像：未来城市
```

**优化方向**: 同通用优化，但输出为中文提示词，适合中文语义理解更强的工具

**输出示例 (中文)**:
```
一只可爱的银渐层猫，蓝眼睛，圆脸，坐在毛毯上，下午柔和自然光，温馨居家氛围，摄影棚布景
```

---

### 3. 中文美学优化

**适用场景**: 中文语境和文化元素

**触发方式**:
```
/image 中国风山水
图像：古装美女
```

**特点**:
- 擅长中文语境
- 文化元素融合
- 国风/传统美学

---

### 3. 摄影向优化

**适用场景**: 摄影作品生成

**触发方式**:
```
/image 人像摄影
图像：夜景风光
```

**强调要素**:
- 主体
- 构图
- 光线
- 氛围
- 无参数/权重语法

---

### 4. 解构/创造性

**适用场景**: 创意、奇幻、独特风格

**触发方式**:
```
/image 创造性：梦境
创意：抽象艺术
```

**特点**:
- 深度解构原始描述
- 跨维度重组
- 创造独特视觉语言
- 保持核心可辨识

---

### 5. JSON结构化

**适用场景**: 需要结构化输出的图像提示词

**触发方式**:
```
/image JSON：猫
图像结构化：城市
```

**输出格式**:
```json
{
  "subject": "主体描述",
  "environment": "环境描述",
  "lighting": "光线描述",
  "composition": "构图描述",
  "style": "风格描述"
}
```

---

## 图生图优化 (image2imageOptimize)

### 1. 通用图生图

**适用场景**: 图像编辑、修改

**触发方式**:
```
图生图：把背景换成夜景
编辑：添加边框
```

**原则**:
- 克制的编辑指导
- 自然语言描述
- 避免参数与权重
- 保持一致性

---

### 2. 设计文案替换

**适用场景**: 保持版式仅替换文本

**触发方式**:
```
文案替换：Hello→你好
设计编辑：换标题
```

**特点**:
- 保持品牌一致
- 保持版式
- 精确文本替换

---

### 3. JSON结构化

**适用场景**: 需要精确控制的图生图

**触发方式**:
```
图生图JSON：人物肖像
编辑结构化：产品图
```

**输出**:
```json
{
  "preserve": ["需要保留的元素"],
  "modify": ["需要修改的元素"],
  "add": ["需要添加的元素"],
  "constraints": ["约束条件"]
}
```

---

## 图像迭代 (imageIterate)

### 通用迭代

**适用场景**: 基于上次结果改进

**触发方式**:
```
图像迭代：让猫更可爱
迭代：增加光影效果
```

**原则**:
- 小步可控
- 保持风格连续
- 视觉意图一致
- 渐进改进

---

## 使用示例

### 示例1: 文生图

**原始**:
> 一只猫

**优化后**:
> 一只可爱的英国短毛猫，圆脸大眼，毛色银灰，柔软光滑。蹲坐在复古木质柜子上，阳光从左侧窗户柔和洒落，在桌面投下温暖的光影。背景是模糊的书架和台灯，营造温馨居家氛围。电影级人像摄影风格，景深效果突出主体。

---

### 示例2: 摄影向

**原始**:
> 城市夜景

**优化后**:
> 上海陆家嘴夜景，摩天大楼群灯光璀璨，东方明珠塔屹立中央。雨后湿润的街道倒映着霓虹灯光，车流形成光轨。赛博朋克未来感风格，蓝色与橙色光线交织。广角镜头，30秒长曝光，三脚架拍摄。

---

### 示例3: 图生图

**原始**:
> 给照片加上黄昏效果

**优化后**:
> 保留：建筑轮廓、人物姿态、主要构图
> 修改：将白天光线调整为黄昏暖色调，天空渐变为橙紫色晚霞
> 添加：阳光穿透云层的耶稣光效果
> 约束：保持原有阴影方向，整体色调统一和谐

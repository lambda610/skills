# Prompt Optimizer

AI 提示词优化工具，帮助你编写高质量的提示词，提升 AI 输出质量。

## 使用方式

直接描述你的需求即可触发：

```
优化：帮我写个文章
系统提示词：你是一个助手
/image 一只猫
评估：
原始提示词：xxx
优化后提示词：xxx
```

## 功能

### 优化用户提示词
```
优化：xxx
```
将模糊、简单的提示词转化为清晰、具体、有效的提示词。

### 优化系统提示词
```
/system xxx
```
将简单的角色描述转化为结构化的系统提示词。

### 迭代改进
```
迭代：
原始提示词或上次优化结果
改进需求：让输出更简洁
```
基于需求二次改进已有提示词。

### 文生图优化
```
/image xxx
```
将自然语言描述转化为高质量的图像生成提示词。

### 图生图优化
描述图像修改需求，生成克制的编辑提示词。

### 评估对比
```
评估：
原始提示词：xxx
优化后提示词：xxx
```
从清晰度、具体性、结构性三个维度评估并给出评分。

## 触发关键词

- `优化：` - 用户提示词优化
- `/system` 或 `系统提示词：` - 系统提示词优化
- `/image` 或 `图像：` - 文生图优化
- `迭代：` - 迭代改进
- `评估：` - 质量评估对比
